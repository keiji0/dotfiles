# Personal

## BetterTouchTool

* [Default.bttpreset](../better-touch-tool/Default.bttpreset)をFinderで開いてプリセットを読み込む
* 設定
  * ベーシック→一般から起動時にBetterTouchToolを起動するにチェック

## Maccy

* 設定
  * `ログイン時に実行`にチェックを入れる
  * 開くのキーボードショートカットを割り当てる

## Stats

* 設定
  * Import settingsから[Stats.plist](../stats/Stats.plist)を読み込む
